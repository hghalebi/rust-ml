# Papers

This directory stores primary-source papers that support the later attention and transformer modules.

## Included Papers

- [Bahdanau et al. (2014): Neural Machine Translation by Jointly Learning to Align and Translate](bahdanau-neural-machine-translation-2014.pdf)
  Original source filename: `1409.0473v7.pdf`
- [Luong et al. (2015): Effective Approaches to Attention-based Neural Machine Translation](luong-effective-approaches-attention-nmt-2015.pdf)
  Original source filename: `Effective Approaches to Attention-based Neural Machine Translation.pdf`
- [Vaswani et al. (2017): Attention Is All You Need](vaswani-attention-is-all-you-need-2017.pdf)
  Original source filename: `NIPS-2017-attention-is-all-you-need-Paper.pdf`

## Usage

- Use Bahdanau and Luong as historical context for pre-Transformer attention.
- Use Vaswani et al. as the canonical Transformer reference.
- Keep lesson prose in `lessons/` and keep source papers here.
