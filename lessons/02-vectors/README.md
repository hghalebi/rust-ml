# 02 Scalars, Vectors, and Dot Products

This folder maps to course Module 1.

This module turns single numbers into structure. It is where ML starts to feel like linear algebra instead of isolated arithmetic.

## Outcomes

After this module, you should be able to:

- distinguish scalars, vectors, and matrices
- compute a dot product and explain what it means
- describe matrix-vector multiplication as repeated row-wise dot products
- explain sigmoid, loss, and the intuition of gradient descent
